# 个人官网

这是一个具有科技感的个人官网，使用HTML、CSS和JavaScript开发。

## 功能特点

- 深科技蓝黑底色
- 蓝紫渐变霓虹光效
- 半透明磨砂玻璃卡片效果
- 卡片悬浮上浮+发光效果
- 按钮渐变放大效果
- 入场淡入动画
- 自适应布局（手机/电脑都能用）

## 页面结构

1. 顶部导航栏
2. 首屏Banner
3. 关于我
4. 核心技能
5. 工作/项目经历
6. 作品集
7. 联系方式
8. 底部

## 如何使用

1. 修改HTML文件中的文字内容
2. 修改CSS文件中的颜色值
3. 替换图片链接
4. 调整卡片宽度、字体大小、间距数值

## 技术栈

- HTML5
- CSS3
- JavaScript
